# Support

## How to Get Help

- **Bug reports and feature requests:** open an issue on [GitHub Issues](https://github.com/moriturus/ghawb.ts/issues).
- **Questions and discussions:** use [GitHub Discussions](https://github.com/moriturus/ghawb.ts/discussions) if enabled, or open an issue.

## Response Expectations

This is a volunteer-maintained open-source project. There is no SLA — responses are provided on a best-effort basis.

## Compatibility Policy

| Runtime | Minimum Version |
| ------- | --------------- |
| Node    | 24+             |
| Bun     | 1.x             |
| Deno    | 2.x             |

## What's in Scope

- Bugs in workflow generation.
- Type system issues.
- Documentation errors.

## What's Out of Scope

- GitHub Actions runtime behavior.
- Custom action development support.
